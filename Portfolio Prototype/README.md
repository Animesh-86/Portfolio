
  # Portfolio Prototype

  A Vite + React + TypeScript portfolio built for Animesh Sharma.

  ## What it includes

  - Animated hero section with a playful code block
  - Project showcase and skills/tech stack sections
  - Certificate gallery with category filters, stacking slider navigation, and click-to-open preview modal
  - Contact section with direct email form handling
  - Framer Aura Night Sky background layered behind the full page

  ## Tech Stack

  - React + TypeScript
  - Vite
  - Tailwind CSS
  - Framer Motion

  ## Running Locally

  Install dependencies:

  ```bash
  npm install
  ```

  Start the dev server:

  ```bash
  npm run dev
  ```

  Build for production:

  ```bash
  npm run build
  ```

  ## Assets

  Certificate images are stored in `public/certificates/`.

  ## Notes

  - The certificate section uses a custom stacking slider with Framer Motion animations.
  - The background effect is rendered as a fixed full-screen layer so the page content can sit above it cleanly.
